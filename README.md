# Shop

### Stack
+ Vue
+ Vuex
+ Vue-router
+ Scss
+ Axios

### 1. Project setup

```
npm install
```

### 2. Compiles and hot-reloads for development

```
npm run serve
```

### 3. Run Database

```
 json-server --watch db.json
```
